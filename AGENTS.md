# Agency Website Working Notes

When building or editing this website, follow [BRAND_GUIDELINES.md](./BRAND_GUIDELINES.md) for Flooencer's visual identity, messaging, imagery, component styling, and copy voice.

Core reminders:
- Use the Flooencer palette, especially `#6513D1`, `#280752`, `#0f0f0f`, `#ffffff`, and `#efefef`.
- Use Degular Display for headings/UI and DM Sans for body copy when available.
- Always use the SVG icon assets from `Icon Updated/icon/` for website UI icons, product diagrams, pills, buttons, navigation, and custom graphics. Do not introduce Lucide, inline SVG, or other third-party icon shapes when the updated icon library has an equivalent.
- The updated icon SVGs are source artwork, not final display sizing. Scale icons up in context: use at least `size-6` for buttons/navigation, `size-7` or `size-8` for card and section icons, and avoid relying on the raw 16x16 SVG default.
- Keep copy direct, proof-led, and oriented around B2B SaaS creator campaigns and pipeline.
- Prefer creator, campaign, pipeline, partner, and brief as the brand vocabulary.
- Avoid generic stock-photo aesthetics and generic enterprise marketing language.
